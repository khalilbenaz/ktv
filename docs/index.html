// EPG maison — API ouverte

Guide sport,
côté serveur

Pour les chaînes sport sans guide fourni, KTV interroge automatiquement un service maison. Tu peux aussi l'appeler directement — JSON, CORS ouvert.

  • GET /sport — grille du jour de toutes les chaînes sport
  • GET /epg?channel=beinsport1 — programme en cours + suivant
  • GET /health — état du service
$ curl "https://ktv-epg.khalilbenaz.workers.dev/epg?channel=beinsport1"

{ "channel":"beinsport1",
  "cur":  {"st":1781915400,"title":"…","en":1781916600},
  "next": {"st":1781916600,"title":"…","en":1781917800} }

Horodatages Unix (UTC), à convertir côté client. Noms normalisés : beinsport1, eurosport2, lequipe… Le parsing se corrige côté serveur, sans mettre à jour l'app.
