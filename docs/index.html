Qualité

Vraie 4K, automatiquement

🎬

Détection 4K réelle

KTV sonde la vraie résolution du flux : seules les chaînes réellement ≥ 2160p sont traitées en 4K (fini les fausses « 4K » 1080p mal étiquetées).

⚡

Transcodage matériel

Les vraies 4K (HEVC 10 bits) sont transcodées par le GPU — VideoToolbox (Mac), NVENC/QSV/AMF (Windows) — pour une lecture fluide.

🧊

Anti-coupure

Tampons ajustés pour absorber les micro-coupures du fournisseur sans figer l'image.

🧠

Mémorisé par chaîne

La décision (4K ou non) est gardée en mémoire : sondée une seule fois, instantanée ensuite.

EPG maison

Comment utiliser l'EPG maison

Dans KTV, c'est automatique : pour les chaînes sport sans guide fourni par l'abonnement, l'app interroge ce service et affiche le programme — rien à configurer. La section ci-dessous s'adresse à ceux qui veulent interroger l'API directement (script, autre app, autre lecteur).

Base : https://ktv-epg.khalilbenaz.workers.dev — réponses JSON (CORS ouvert), grille rafraîchie ~toutes les 30 min (source : programme-tv).

  • GET /sport — grille du jour de toutes les chaînes sport (beIN Sports 1/2/3, Canal+ Sport, Eurosport, L'Équipe…). Renvoie { day, tz, channels: { <chaîne>: [ { st, en, title } ] } }.
  • GET /epg?channel=beinsport1 — programme en cours + suivant d'une chaîne : { channel, cur:{st,title,en}, next:{st,title,en} }.
  • GET /health — état du service : { "ok": true }.
# Programme en cours et suivant de beIN Sports 1
curl "https://ktv-epg.khalilbenaz.workers.dev/epg?channel=beinsport1"

# → {"channel":"beinsport1",
     "cur": {"st":1781915400,"title":"…","en":1781916600},
     "next":{"st":1781916600,"title":"…","en":1781917800}}

st / en = horodatages Unix en secondes (UTC) — à convertir dans le fuseau de l'utilisateur côté client. Les noms de chaînes sont normalisés (minuscules, sans accents/espaces) : beinsport1, eurosport2, canalsport360, lequipe… Avantage : le parsing se corrige côté serveur, sans mettre à jour l'app.

Installer

Installation

  1. macOS — décompresse KTV-macos-arm64.zip.
  2. Glisse KTV dans le dossier Applications.
  3. Clic-droit → Ouvrir (app non notarisée — une seule fois).
  4. Si bloqué : xattr -dr com.apple.quarantine "/Applications/KTV.app"
  1. Windows — décompresse KTV-windows-x64.zip.
  2. Lance KTV.exe.
  3. Si SmartScreen apparaît : Informations complémentaires → Exécuter quand même.

macOS Apple Silicon (M1/M2/M3+) · Windows x64. Abonnement à 1 connexion : lecture, enregistrement et restream partagent un relais local unique. Binaires construits automatiquement via GitHub Actions.
