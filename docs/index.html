Installation

  1. macOS — décompresse KTV-macos-arm64.zip.
  2. Glisse KTV dans le dossier Applications.
  3. Clic-droit → Ouvrir (app non notarisée — une seule fois).
  4. Si bloqué : xattr -dr com.apple.quarantine "/Applications/KTV.app"
  1. Windows — décompresse KTV-windows-x64.zip.
  2. Lance KTV.exe.
  3. Si SmartScreen apparaît : Informations complémentaires → Exécuter quand même.
